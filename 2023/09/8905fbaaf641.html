    The theme of my report is **Visual Analysis of Enron Email Network。

    This is the table of contents for this report, including Dataset Introduction
    , Data Processing Workflow, Network Attribute Statistics and Visualization Based on Gephi.

    1 Dataset Introduction

    Now let’s move on to the first part.

    The Enron email dataset refers to the email records of the Enron Corporation prior to its bankruptcy in 2001 due to an accounting scandal. These email records became crucial evidence, revealing financial fraudulent activities within the company. Today, this email dataset has become a classic case in the fields of data analysis and machine learning. The dataset represents email users as nodes, and each email correspondence as edges.

    This image shows the statistical information obtained from the SNAP website for this dataset:

    The network has 36692 nodes and 18381 edges.

    2 Data Processing Workflow

    The second part is Data Processing.

    In this step, we will convert the txt format dataset downloaded from the Snap website to gexf format.
    At the same time, initialize the network using the methods in networkx.

    3 Network Attribute Statistics

    Next is the third part of Network Attribute Statistics

    The network related properties calculated using networkx are as follows:
    The proportion of existing edges to the maximum possible number of edges is  0.027309755503535.
    The Average_Degree is 10.020222391802028 And the Diameter is 13.
    On the right, there are network related attributes that use gephi Statistics.
    We can see that the data obtained by two different methods are relatively close.

    On the left is the degree distribution curve drawn using networkx, while on the right is the curve obtained using gephi.
    We can see that the degree distribution of this network follows the characteristics of scale-free networks.
    This picture is the result of taking the logarithm of the horizontal and vertical coordinates of the above figure.
    It can be seen that the network has a large number of small degree nodes.

    The fourth part is visualization based on gphi

    Layout

    First of all, choose Yifan Hu layout for adjustment.
    The Yifan Hu layout algorithm can quickly process graphs with hundreds of thousands of node sizes and has good visualization effects on specific networks with a large number of nodes.

    Community-detection

    To further analyze the characteristics of the network, we will modularize the network.

    The modularity obtained from gephi is 0.67, indicating that the network has a high degree of modularization.

    The following figure shows the result of modularization and color readjustment

    Filter

    From the modularized graph, it can be seen that the network is mainly composed of different communities diverging from several large degree nodes
    Therefore, when filtering, we use a filtering method called modularity class to purify the main communities.

    Firstly, there are some display images of individual partitions
    Next is the effect image of merging several major communities

    Preview

    Finally, there is the final visualization of the Enron email network.

    From the graph, it can be seen that in the Enron incident, there was a close connection between the senior management of different departments, but there was less email communication between the majority of employees in each department, and there were a large number of groups within each department.
